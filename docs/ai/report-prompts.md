## Использованные промпты

- Промпт 1 (основной запрос на генерацию AI-инструкций):  
  «Ты — опытный инженер-наставник и эксперт по работе с AI-инструментами. Твоя задача — создать полный набор AI-инструкций для проекта в Cursor, включая: файл .cursorrules ... Убедись, что: .cursorrules готов к размещению в корне проекта; документация в docs/ai/ дополняет правила из .cursorrules; инструкции позволяют агенту работать без дополнительных уточнений.»

- Промпт 2 (пример для CLI-опции `--max-fps`):  
  «Добавь в C-клиент scrcpy новую CLI-опцию `--max-fps=<N>` для ограничения максимального FPS. Опиши план действий, укажи конкретные файлы в `app/`, измени парсер аргументов и логику ограничения FPS. Обнови соответствующую документацию в `doc/video.md` с примером команды запуска. В конце приведи команды сборки и шаги проверки.»

- Промпт 3 (пример для Android-сервера):  
  «Нужно расширить Android-сервер scrcpy для поддержки новой опции видео (например, особого режима битрейта). Определи, какие классы в `server/src/main/java/com/genymobile/scrcpy/video/` нужно изменить, обнови протокол обмена с C-клиентом по необходимости и опиши влияние на совместимость. Обязательно спроси подтверждение перед изменением протокола. В конце перечисли шаги сборки (`./gradlew`) и ручной проверки.»

- Промпт 4 (пример для документации TCP/IP):  
  «Улучшить документацию по подключению scrcpy по TCP/IP в файле `doc/connection.md`. Сделай структурированный раздел с пошаговыми командами, примером успешного сценария и типичными ошибками. Сохрани текущий стиль документа и не ломай существующие ссылки. Приведи итоговый текст раздела и кратко перечисли изменения.»
